# Network-setup
Здесь собрана информация по самостоятельной настройке сетевых устройств и сервисов.

## Оглавление

- **[Настройка самоподписных сертификатов](./certs/)**  
  Инструкция по SSL для LuCI, uHTTPd и других сервисов
- **[Настройка Kuma](./kuma)**  
  Инструкция по настройке и использованию Kuma для отслеживания состояния сети
- **[Настройка Adguard Home](./adguard)**  
  Инструкция по настройке Adguard Home и использвание его как основного DNS резолвера
