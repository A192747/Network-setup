# AdGuard Home на OpenWrt

Инструкция по установке и настройке AdGuard Home на роутере с прошивкой OpenWrt.

## Требования

- Роутер с установленной прошивкой OpenWrt
- Установленный Docker и Docker Compose
- Свободное место на накопителе для работы контейнера

## Установка

### 1. Подготовка директории

Создайте директорию для AdGuard Home и скопируйте в неё файлы конфигурации:

```bash
mkdir -p /path/to/adguard
cd /path/to/adguard
```

### 2. Запуск контейнера

Запустите AdGuard Home с помощью Docker Compose:

```bash
docker-compose up -d
```

### 3. Первоначальная настройка

1. Откройте веб-интерфейс AdGuard Home в браузере: `http://<IP-роутера>:3000`
2. Пройдите мастер первоначальной настройки
3. Настройте фильтры и правила блокировки рекламы

### 4. Настройка DNS на роутере

После завершения настройки AdGuard Home выполните скрипт для удаления других DNS-серверов:

```bash
./remove-other-dns-servers.sh
```

**Важно:** Этот скрипт удаляет все сторонние DNS-серверы из конфигурации dnsmasq и устанавливает единственный DNS-провайдер — AdGuard Home (на порту 5055). Это гарантирует, что весь DNS-трафик будет проходить через AdGuard Home.

Скрипт выполняет следующие действия:
- Удаляет все настроенные DoH-серверы (`doh_server`, `doh_backup_server`)
- Удаляет все стандартные DNS-серверы (`server`)
- Добавляет единственный сервер `127.0.0.1#5055` (AdGuard Home)
- Перезапускает службу dnsmasq

1. Откройте веб-интерфейс AdGuard Home

3. Добавьте URL нужного фильтра

## Структура файлов

```
adguard/
├── docker-compose.yml          # Конфигурация Docker Compose
├── remove-other-dns-servers.sh # Скрипт настройки DNS
├── work/                       # Рабочие данные AdGuard Home
└── conf/                       # Файлы конфигурации AdGuard Home
```

## Порты

| Порт | Назначение |
|------|------------|
| 5055 | DNS-сервер (TCP/UDP) |
| 8000 | Веб-интерфейс (после настройки) |
| 3000 | Мастер первоначальной настройки |

## Проверка работы

Убедитесь, что DNS-запросы обрабатываются через AdGuard Home:

```bash
nslookup example.com 127.0.0.1 -p 5055
```

Или проверьте логи контейнера:

```bash
docker logs adguardhome
```

## Обновление

Для обновления AdGuard Home до последней версии:

```bash
docker-compose pull
docker-compose up -d
```
